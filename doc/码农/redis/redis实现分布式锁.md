2.项目中是怎么使用分布式锁的？
官方推荐采用Redlock算法，即使用string类型，加锁的时候给的一个具体的key，然后设置一个随机的值；
取消锁的时候用使用lua脚本来先执行获取比较，然后再删除key。
加锁 :
      SET resource_name my_random_value NX PX  30000
取消锁:
    if redis.call( "get" ,KEYS[ 1 ]) == ARGV[ 1 ] then
    return redis.call( "del" ,KEYS[ 1 ])
    else
    return  0
    end
